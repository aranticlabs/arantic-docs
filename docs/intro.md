## A note on AI limitations

AI models make mistakes. They hallucinate APIs, misunderstand context, and produce code that looks correct but isn't. Treat AI output as a **first draft**, not a finished product. Always review, test, and understand the code before using it in production. For common pitfalls and how to handle them, see [Troubleshooting & Limitations](/resources/troubleshooting).
