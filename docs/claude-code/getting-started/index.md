# Get Started with Claude Code

Pick the guide that matches your experience level. Each one builds on the previous, so you can start wherever you are and work forward.

| Level | Who it's for | What you set up | Time |
|-------|-------------|-----------------|------|
| [**Starter**](getting-started/starter) | Never used AI coding tools | Install, CLAUDE.md, basic commands | ~15 min |
| [**Intermediate**](getting-started/intermediate) | Already built projects with Claude Code | Memory, permissions, skills, subagents, MCP | ~30 min |
| [**Pro**](getting-started/pro) | Wants full automation and orchestration | Hooks, agent teams, CI/headless, workflows | ~45 min |

## Not sure where to start?

If you've never run `claude` in a terminal before, start with the **Starter** guide. If you already have Claude Code installed and have used CLAUDE.md files, jump to **Intermediate**. If you're already using custom skills, MCP servers, and subagents, head straight to **Pro**.

Looking for IDE-specific setup instead? Check the stack-based [Quickstart](/quickstart) guides for Web, .NET, and firmware projects.
